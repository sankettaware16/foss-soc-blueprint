# FOSS SOC Blueprint

This repository explains how organizations can design and build a complete
Security Operations Center (SOC) using Free and Open-Source Security Tools.

It is written from the perspective of a Security Engineer who focuses on
practical SOC architecture, log visibility, and detection engineering.

---

## Who Is This For?

- Startups
- Universities
- NGOs
- Small & Medium Businesses (SMBs)
- Teams with limited security budgets

---

## What This Repository Covers

- SOC architecture using FOSS tools
- Log sources and normalization strategy
- Detection engineering fundamentals
- Alert triage and incident response
- Production deployment guidance
- Cost comparison: FOSS vs commercial SOC tools

---

## Tools Covered

- Elasticsearch, Logstash, Kibana (ELK Stack)
- Wazuh
- Suricata
- Zeek
- TheHive + MISP
- Osquery

---

## Why FOSS SOC?

Commercial SOC platforms are often expensive, complex, and vendor locked.

A properly designed FOSS SOC can provide strong security visibility and
detection capability at minimal cost.
